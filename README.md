# Features

Este projeto tem por objetivo exemplificar um servidor e um cliente fix, foi feito o setup do projeto para executar e conectarem entre si sem necessidade de ajustes.

# Versões compiladas

Na pasta exec há versões compiladas para windows, favor utilizar os comandos abaixo para execução dos mesmos.

```
 FixProject.Client.exe ".\conf\client.cfg"
 FixProject.Server.exe ".\conf\server.cfg"
```
